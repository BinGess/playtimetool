# Bio-Detector 设计文档

## 目标
新增一个“生物检测器（Bio-Detector）”聚会游戏，以“伪生物检测 + 心理欺骗”为核心体验。游戏不读取任何真实生物传感数据，所有检测过程为视觉/震动/文案伪装。

## 核心流程
1. 待机：用户在指纹区域长按。
2. 初始化：显示 `Initializing Bio-Link...`，触发短促“电流感”微震。
3. 采样阶段（0s-5s）：红色滚动脉搏波 + 指标读取滚动文案。
4. 高压阶段（5s-10s）：震动频率从 60bpm 线性提升至 120bpm，随机黄字警告闪烁。
5. 结果页：
   - TRUTH：绿色圆环 + `TRUTH: 98%`，震动平缓消失。
   - LIE：红色惊叹号 + `LIE DETECTED`，连续三次短促强震。

## 欺骗操纵杆（隐藏逻辑）
- 透明热区：屏幕左上角 44x44pt、右下角 44x44pt。
- 映射：左上角触发强制 TRUTH；右下角触发强制 LIE。
- 若未触发任何热区：结果按 50/50 随机。

## 视觉风格
- 背景：深灰示波器风格 + 10px 网格。
- 扫描线：半透明扫描光栅 2s 周期上下扫动。
- 扫描线扫过指纹区域时，出现局部高亮。

## 技术约束
- 分层实现：界面层负责动画与交互，逻辑层负责阶段/结果决策。
- 保持与现有 Hub、路由、本地化和帮助弹窗体系一致。
- 页面销毁时必须停止计时器与动画控制器。
